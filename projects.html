Celestes CCG

1 year project

Celestes is an online multiplayer collectible card game (CCG) set in a fantasy universe. The game features strategic gameplay where players collect and build decks of cards, each representing characters, effects, and abilities. Key aspects include gameplay mechanics: players engage in tactical battles using their customized decks, requiring strategic planning and foresight. Card Variety: A diverse range of cards allows for extensive deck customization and varied playstyles. Art and Design: High-quality artwork enhances the visual appeal and immersion of the game. Community Engagement: daily and weekly rewards keep the player community active and engaged. Available @ www.celestesccg.com

Turtle Beach

1 month project

Action game where player controls a character with different upgradable skills and attributes to ward off predators so that baby turtles make it to the ocean safely. Game features multiple progress saving and loading slots, playful cartoon movement and animation, cue sound effects, multiple levels of increasing difficulty, reward system based on performance, upgrades shop, multiple enemies and final boss. Available @ www.lucasferraripereira.itch.io/turtle-beach
